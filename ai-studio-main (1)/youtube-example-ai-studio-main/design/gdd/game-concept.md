> бездомным котам. Каждое прикосновение — это маленькая терапия: для кота и для вас.

## Core Fantasy

Вы — хранитель маленького приюта. Каждый кот приходит с улицы напуганным и
недоверчивым. Через ваше терпение и заботу они раскрываются, становятся
счастливыми, и однажды находят свой настоящий дом. А вы расширяете приют, чтобы
спасти ещё больше.

**Эмоциональное обещание:** ощущение тёплой, тихой заботы — каждое поглаживание
имеет значение.

---

## Unique Hook

Как Neko Atsume, **и ещё** каждый кот — это тактильная головоломка: у каждого свой
характер, любимые зоны поглаживания, настроение и история. Глажка — не просто клик,
а полноценное физическое взаимодействие с feedback-ом.

## Player Experience Analysis (MDA Framework)

### Target Aesthetics (What the player FEELS)

| Aesthetic | Priority | How We Deliver It |
| ---- | ---- | ---- |
| **Sensation** (sensory pleasure) | 2 | Тактильный feel глажки, мурчание, сочные анимации |
| **Fantasy** (make-believe, role-playing) | 3 | Уникальные коты с историями и характерами |
| **Narrative** (drama, story arc) | 6 | Мини-истории каждого кота, bitter-sweet прощания |
| **Challenge** (obstacle course, mastery) | N/A | Намеренно избегаем стресса |
| **Fellowship** (social connection) | N/A | Single-player, связь с котами заменяет social |
| **Discovery** (exploration, secrets) | 5 | Новые породы, редкие коты, скрытые предпочтения |
| **Expression** (self-expression, creativity) | 4 | Декорирование приюта, выбор как ухаживать |
| **Submission** (relaxation, comfort zone) | 1 | Медленный темп, ambient звуки, нет fail-state |

### Key Dynamics (Emergent player behaviors)

- Игроки будут экспериментировать с зонами поглаживания, находя "любимые точки" каждого кота
- Игроки привяжутся к конкретным котам и будут откладывать их "выпуск" домой
- Игроки будут стремиться собрать фотостену всех "выпускников"
- Игроки будут возвращаться вечером как к ритуалу расслабления

### Core Mechanics (Systems we build)

1. **Petting System** — тактильное взаимодействие мышкой с зонами на теле кота, реакции и feedback
2. **Cat Happiness** — метр счастья, зависящий от ухода, кормления и поглаживания
3. **Shelter Economy** — донаты посетителей, расширение приюта, покупка предметов ухода
4. **Cat Personality** — уникальные характеры, предпочтения, истории, реакции
5. **Visitor System** — посетители приходят, взаимодействуют со счастливыми котами, донатят или усыновляют

### Primary Psychological Needs Served

| Need | How This Game Satisfies It | Strength |
| ---- | ---- | ---- |
| **Autonomy** (freedom, meaningful choice) | Выбор каких котов гладить, как украшать приют, когда отпускать кота домой | Supporting |
| **Competence** (mastery, skill growth) | Изучение предпочтений котов, эффективный уход, рост приюта | Supporting |
| **Relatedness** (connection, belonging) | Эмоциональная привязанность к котам, их истории, прощания | Core |

### Player Type Appeal (Bartle Taxonomy)

- [x] **Achievers** (goal completion, collection, progression) — Коллекция выпускников, расширение приюта, разблокировка предметов
- [x] **Explorers** (discovery, understanding systems, finding secrets) — Открытие предпочтений котов, редкие породы, скрытые механики
- [x] **Socializers** (relationships, cooperation, community) — Привязанность к котам как к персонажам, их истории
- [ ] **Killers/Competitors** — Не наша аудитория

### Flow State Design

- **Onboarding**: Первые 10 минут — один кот, обучение глажке, первый "успех" (кот замурчал)
- **Difficulty scaling**: Новые коты сложнее в характере (пугливые, капризные), но нет fail-state
- **Feedback clarity**: Визуальные сердечки, мурчание, happiness-бар, реакции кота
- **Recovery from failure**: Нет провала — кот просто отворачивается, можно попробовать снова

## Core Loop

### Moment-to-Moment (30 seconds)

Гладите кота мышкой — курсор превращается в руку, вы водите по коту. Каждый кот
реагирует по-разному: один любит за ушком, другой — под подбородком, третий — по
спинке. Правильное поглаживание = мурчание, сердечки, рост happiness-бара.
Неправильное = кот отворачивается (но без наказания).

Это ДОЛЖНО быть приятно само по себе — без наград, без прогрессии, просто feel.

### Short-Term (5-15 minutes)

Обходите приют, проверяете котов. Кто голоден — покормить. Кто грустит — погладить.
Кто готов — показать посетителю. Посетители заходят, смотрят на счастливых котов,
оставляют донаты или забирают кота домой (bitter-sweet момент).

### Session-Level (30-60 minutes)

Цикл дня: Утро -> День -> Вечер.
- **Утро**: приходят новые уличные коты (событие)
- **День**: посетители и уход (основной геймплей)
- **Вечер**: тихий момент, итоги дня, планирование расширения

Естественная точка остановки в конце каждого дня.

### Long-Term Progression

Копите деньги -> расширяете приют (новые комнаты, двор, игровая зона) -> помещается
больше котов -> больше посетителей -> больше донатов. Открываете новые предметы ухода,
игрушки, декор. Коллекция "выпускников" — фотостена котов, нашедших дом.

### Retention Hooks

- **Curiosity**: Какой кот придёт завтра? Какой характер у нового?
- **Investment**: Привязанность к текущим котам, прогресс приюта
- **Social**: Нет (single-player), заменяется эмоциональной связью с котами
- **Mastery**: Научиться быстро находить подход к сложным котам

## Game Pillars

### Pillar 1: Каждое прикосновение имеет значение

Глажка — ядро всей игры. Она должна быть тактильно приятной, отзывчивой и
эмоционально наполненной. Никакого автоклика, никакой автоматизации ухода.

*Design test*: "Добавить автоматическую кормушку?" — Нет, ручное взаимодействие
важнее эффективности.

### Pillar 2: Котики — личности, не ресурсы

Каждый кот уникален: имя, характер, история, предпочтения. Игрок должен
привязываться к ним. Коты — не конвертируемые единицы в экономике.

*Design test*: "Сделать котов процедурно-генерируемыми шаблонами?" — Нет, каждый
должен чувствоваться особенным.

### Pillar 3: Уют без стресса

Нет game over, нет таймеров, нет наказаний. Игрок всегда в безопасности. Темп
задаёт игрок, а не игра.

*Design test*: "Добавить штраф за голодных котов?" — Нет, коты грустнеют, но
никогда не умирают и не убегают.

### Anti-Pillars (What This Game Is NOT)

- **НЕ менеджмент-тайкун**: мы не строим оптимизатор прибыли. Деньги — средство заботы, не цель.
- **НЕ гринд**: прогрессия через эмоциональные моменты, не через числа и повторение.
- **НЕ сложная игра**: нет fail-states, нет pressure, нет competitive-элементов.

## Risks and Open Questions

### Design Risks
- Глажка может наскучить — нужен отличный tactile feel и вариативность котов
- Без challenge-а сессии могут не удерживать 30+ минут

### Technical Risks
- Физика поглаживания (реакция кота на движение курсора) — ключевой R&D
- Анимационная система должна плавно смешивать реакции кота

### Market Risks
- Cozy-жанр растёт, но конкуренция высокая — нужен сильный арт и уникальный feel
- "Просто коты" может быть недостаточно для удержания без сильного hook-а

### Scope Risks
- 15-20 уникальных котов с индивидуальными анимациями — большой объём контента
- Hand-drawn арт-стиль трудоёмок для соло/малой команды

### Open Questions
- Насколько детальной должна быть система глажки? Прототип ответит.
- Bitter-sweet прощание с котом — как сбалансировать грусть и радость?
- Нужен ли цикл день/ночь или свободное время лучше?

## MVP Definition

**Core hypothesis**: "Игрокам приятно гладить котов в течение 15+ минут благодаря
тактильному feedback-у и уникальным реакциям каждого кота."

**Required for MVP**:
1. Одна комната приюта с 3 котами
2. Система глажки с зонами и реакциями (мурчание, сердечки, отворачивание)
3. Базовый happiness-метр
4. Один тип посетителя с донатами

**Explicitly NOT in MVP** (defer to later):
- Расширение приюта
- Коллекция выпускников / фотостена
- Декорирование приюта
- Цикл день/ночь
- Усыновление котов

### Scope Tiers

| Tier | Content | Features | Timeline |
| ---- | ---- | ---- | ---- |
| **MVP** | 1 комната, 3 кота | Глажка + happiness + донаты | 3-4 недели |
| **Vertical Slice** | 2 комнаты, 5 котов | + кормление, посетители, 1 расширение | 6-8 недель |
| **Alpha** | 4 комнаты, 10 котов | + цикл дня, усыновление, декор | 3-4 месяца |
| **Full Vision** | 7 зон, 20 котов | Всё + фотостена, редкие коты, полный контент | 5-6 месяцев |
